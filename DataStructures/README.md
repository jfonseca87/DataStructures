# DataStructures

A .NET 6 console application that explores fundamental data structures in C# through practical, real-world examples. Each data structure is demonstrated with minimal code snippets showing how to create, populate, and interact with it.

The project covers **Stacks** (LIFO - browser back button analogy, word reversal), **Queues** (FIFO - print queue analogy), **Linked Lists** (doubly-linked list navigation), **Graphs** (social network friendships and package dependency graphs using adjacency lists), and **Trees** (file system hierarchy, binary search trees, and organizational employee hierarchies). Each example is wrapped in its own method and can be uncommented in `Main()` to run individually.

## Technologies

- .NET 6
- C#
- Console Application

## Features

- Stack (Push, Pop, Peek, word reversal)
- Queue (Enqueue, Dequeue)
- LinkedList (node navigation)
- Graph (social network adjacency list, package dependency graph)
- Tree (file system tree, binary search tree, organizational hierarchy)

## How to Run

```bash
dotnet run --project DataStructures
```

Uncomment the desired example method call in `Program.cs` to test each data structure.
